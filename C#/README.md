# Alarm Clock Project
Basic functionalities:
- Set Alarm
- Snooze for 10 seconds
- Timer/Stopwatch [1]

**[1] Timer will be deducted by 10 seconds.** <br/>
Developed for the game Runescape as being away for more than 5 minutes will cause the player to log out.
